## Overview

Kudos Wall is a team appreciation platform built during the Avesta AI Hackathon, where it secured the **2nd Prize**. The platform allows team members to post, browse, and manage kudos, fostering a positive work culture. It was built in just 48 hours with a focus on clean architecture and modular design.

## Key Features

- **Kudos Management**: Core features for posting, browsing, and managing kudos.
- **Responsive UI**: A user-friendly interface that works seamlessly across devices.
- **Clean Architecture**: Backend designed using Clean Architecture principles for maintainability and scalability.
- **TDD Approach**: Followed Test-Driven Development throughout the development process.

## Tech Stack

- **Backend**: Node.js, TypeScript
- **Database**: MongoDB
- **Frontend**: Next.js

## Outcome

Delivered a fully functional and polished platform within a tight 48-hour deadline, demonstrating the ability to build high-quality software under pressure. The project was recognized for its clean code and modular architecture.
